CONCEPTO DE HTML5

Para entender las etiquetas de HTML5, entraremos primero a conceptualizar de que se trata este lenguaje y además como ha sido el desarrollo a través de la historia. La sigla HTML (Terán Anciano, 2016) se refiere a (Hyper Text Markup Language), el cual es un lenguaje de programación que utiliza códigos llamados etiquetas, que bien organizadas podemos realizar la construcción de una página web que contiene texto, imagen, títulos, enlaces, etc.

A continuación, la historia del HTML que publica (Celaya, 2014) en su libro de Creación de páginas web: HTML5 resumida a su máxima expresión:

- Año 1980, el físico Tim Berners-Lee crea HTML en 1980, un sistema de “hipertexto” con el fin de compartir documentos en la organización CERN (Organización Europea para la Investigación Nuclear) para la que trabajaba en ese momento.

- Año 1991, HTML primera publicación

- Año 1993, definen etiquetas para imágenes, tablas y formularios.

- Año 1995, HTML 2.0. orientado a la actividad académica, su contenido era mas importante que el diseño.

- Año 1997, en el mes de enero aprueban el estándar HTML 3.2. mejoras del lenguaje por parte de Internet Explorer y Netscape Navigator.

- Año 1997, en el mes de diciembre aprueban el estándar HTML 4.0. versión para estandarizar los marcos, las hojas de estilo y los scripts.

- Año 2004, fabricantes de navegadores forman un grupo WHATWG (Web Hypertext Application Technology Working Group) para crear un lenguaje mejor orientado a nuevo tipo de aplicaciones, pero manteniendo con la compatibilidad de los navegadores existentes

- Año 2006, Tim Berners-Lee anunció alianza con W3C y WHATWG para la elaboración del nuevo estándar HTML5

- Año 2010, W3C presenta el logo HTML5 que se incorporan en las páginas que utilizan esta tecnología.
